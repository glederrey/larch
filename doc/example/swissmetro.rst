
======================
Swissmetro Examples
======================

The Swissmetro sample dataset is the same data used in the examples for `Biogeme <http://biogeme.epfl.ch/>`_.
Note that Larch and Biogeme have different capabilities, and it is not possible to reproduce every example
model shown for Biogeme.
